# anime-video-production

AI 漫剧制作系统 · 即梦 Seedance 2.0 全流水线 Claude Code Skill

## 功能

完整覆盖 AI 漫剧从创意到成片的全部环节：

```
规划 → 剧本 → 台词 → 素材生成 → 视频提示词 → 转场 → 合集拼接
```

### 核心机制

- **真相文件系统** — 持久化追踪世界观/时间线/素材状态，防止角色漂移和场景穿越
- **三层规则栈** — 通用护栏(16条) → 视觉风格(2D日漫/3D国漫) → 项目专属规则，每集生成前自动注入
- **集数门禁(4道)** — 门禁0 剧本一致性+台词合理性 → 门禁1 素材完整性 → 门禁2 提示词合规(提交前拦截) → 门禁3 时间线验证
- **8步 Agent 流水线** — 规划→剧本→台词→素材→视频→结算→转场→拼接，每步有明确输入输出和质量关卡

### 剧本与台词系统

- **剧本先行** — 先写完整剧本(场景/行为/情绪/冲突)，再生成台词，最后填入提示词。三步分离，禁止提示词阶段现编台词
- **上下文连续性** — 剧本Agent 写作前必须读取前1-2集剧本和台词表，保证跨集剧情衔接、伏笔呼应、反派级别递增
- **章节边界检查** — 自动验证本集反派级别是否匹配 series_bible.md 中该章定义的对手范围
- **跨集重复检查** — 台词表写完后扫描与前2集的标志性台词/句式是否雷同

### 题材台词风格库

按题材分类的台词写作模板，台词Agent 根据项目声明的题材加载对应风格：

| 题材 | 状态 |
|------|------|
| 玄幻逆袭·爽文向 | ✅ 已完成 |
| 都市高武·热血向 | 📋 预留框架 |
| 甜宠·言情向 | 📋 预留框架 |
| 悬疑脑洞·规则怪谈向 | 📋 预留框架 |
| 历史权谋·古风向 | 📋 预留框架 |
| 末世生存·废土向 | 📋 预留框架 |

**玄幻逆袭核心规则：**
- 文风DNA = 套番茄/起点网文壳子，台词 = 网文金句视觉化
- 台词核心结构 = "对比碾压三段式"（你当年怎么对我 → 你今天算什么 → 反差即爽感）
- 独白是主力对话形式（不受6字限制·需要配音·承载金句），口型对白是点睛（≤6字·有对象感）
- 每集至少1句金句级台词

## 使用

在 Claude Code 中输入 `/anime-video-production` 即可启动 AI 漫剧导演模式。

### 快速开始

1. 初始化项目：AI 自动生成真相文件模板（series_bible.md / book_rules.md / project.json 等）
2. 规划：输入故事大纲，AI 生成分镜大纲和素材需求
3. 剧本→台词：AI 按流水线生成剧本和台词表，自动通过门禁检查
4. 素材生成：`dreamina text2image` 角色立绘 + 场景图
5. 视频生成：AI 生成提示词 → 门禁扫描 → `dreamina multimodal2video` 提交
6. 结算转场拼接：截取首尾帧 → 生成转场 → ffmpeg 合成

### 项目文件结构

```
项目根目录/
├── series_bible.md           ← 系列圣经（世界观·角色·章节结构）
├── book_rules.md             ← 项目规则（题材声明·外形锚点·台词规则）
├── current_focus.md          ← 当前创作焦点
├── project.json              ← 素材映射
├── timeline_truth.json       ← 时间线状态
├── asset_truth.json          ← 素材生成状态
├── gate_results/             ← 门禁结果
├── 素材/角色/ · 场景/        ← 角色立绘·场景图
├── 提示词/剧本/ · 视频/      ← 剧本·台词表·视频提示词
├── 输出/                     ← 视频成片
├── 帧图像/                   ← 首尾帧截图
└── 任务记录/                 ← submit_id 记录
```
