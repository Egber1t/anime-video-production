# anime-video-production

AI 漫剧制作系统 · 即梦 Seedance 2.0 + 外部 TTS 配音 · 混合工具链 Claude Code Skill

## 定位

面向快节奏爽剧（玄幻逆袭/都市高武/甜宠言情）的 AI 漫剧量产系统。

**核心生产顺序：先配音 → 再让画面跟着音频走。**

即梦负责所有"画面生成"，外部工具负责所有"声音层"。

## 功能

完整覆盖 AI 漫剧从创意到成片的全部环节：

```
规划 → 剧本 → 台词 → 配音 → 素材生成 → 视频提示词 → 口型同步 → 转场 → 合集拼接
```

### 核心机制

- **真相文件系统** — 持久化追踪世界观/时间线/素材状态/角色状态，防止漂移和穿越
- **三层规则栈** — 通用护栏 → 视觉风格(2D日漫/3D国漫) → 项目专属规则，每集生成前自动注入
- **集数门禁(4道)** — 门禁0 剧本一致性+台词合理性 → 门禁1 素材完整性 → 门禁2 提示词合规+审核规避 → 门禁3 时间线验证
- **9步 Agent 流水线** — 规划→剧本→台词→**配音**→素材→视频→结算→**口型同步**→转场→拼接

### 分镜导演规范（与即梦官方提示词结构对齐）

每个15秒时间块必须覆盖13个维度：

```
主体(关系描写) → 姿态 → 表情 → 景别 → 视角 → 构图 → 运镜 → 动态控制 → 角色动作 → 物品/特效 → 环境动态 → 时间 → 氛围
```

- **景别**：6级（极远景/全景/中景/近景/特写/大特写），每15秒≥2种
- **视角**：4种（低角度/高角度/平视/鸟瞰）
- **构图**：5种（对角线/三分/对称/越肩/荷兰角）
- **运镜**：15种官方基础运镜 + 叠加运镜（高潮时间块可用）
- **动态控制**：小/中/大动态三级，高潮必须用大动态
- **动作速度分层**：基础层(匀速缓慢) → 强调层(慢动作特写) → 爆发层(极慢蓄力→瞬间释放) → 连击层(顿点式拆解)

### 台词三层分类体系（高密度爽文模式）

| 类型 | 定义 | 口型同步 | 画面要求 | 占比 |
|------|------|---------|---------|------|
| **核心对白** | 情绪高潮/Hook/关键剧情 | **必须做** | 正面特写/近景 | ≤30% |
| **字幕台词** | 信息补充/过渡 | 不做 | 字幕气泡闪现 | 50-60% |
| **旁白** | 画外音叙述 | 不做 | 空镜/背影/侧脸 | 10-20% |

**爽文台词密度：** 每段80-120字，全集约400-600字。全程有声音推进，打斗段也用旁白解说。

**节奏模板：** 短句叠加 / 排比递进 / 先抑后扬 / 反问碾压

### 融图审核规避

即梦内容审核严格，提示词必须替换敏感词：

- **在世艺术家**：宫崎骏→吉卜力风格，新海诚→删除人名保留效果
- **血液/伤害**：血液→暗红液体，杀/死→消散/崩解/化为光点
- **爆炸/暴力**：爆炸→能量爆发/光柱冲天
- **绝对禁止**：毒品/虐待/恐怖主义/涉政/涉军/商标IP/裸露

### TTS 配音与口型同步（先配音后画面）

```
❌ 错误：写台词 → 生成视频 → 贴配音 → 口型对不上 → 重抽
✅ 正确：写台词 → TTS生成配音 → 音频驱动口型 → 合成
```

**TTS 工具推荐：**
- **海螺AI** — 中文情绪还原最好，玄幻国漫首选
- **Fish Audio** — 音色克隆精度高，支持角色专属声线
- **ElevenLabs** — 多语言混合，英文最强

**口型同步工具：**
- **可灵** — 对口型效果较好，推荐
- **即梦网页端** — 自带对口型，效果不稳定
- **HeyGen** — 数字人口型极精准，成本高

**口型同步决策树：**
```
核心对白 + 正面特写/近景 → 必须做口型同步
核心对白 + 侧面/背影/大角度 → 换分镜为正面再做
字幕台词 → 字幕气泡，不做口型
旁白 → 切空镜/背影/侧脸，不做口型
```

## 使用

在 Claude Code 中输入 `/anime-video-production` 即可启动 AI 漫剧导演模式。

### 快速开始

1. **初始化项目**：AI 自动生成真相文件模板（series_bible.md / book_rules.md / project.json 等）
2. **规划**：输入故事大纲，AI 生成分镜大纲和素材需求
3. **剧本→台词**：AI 按流水线生成剧本和台词表（含三层分类标注），自动通过门禁0
4. **配音**：外部 TTS 生成音频文件 + 配音标注表（情绪/语速/停顿）
5. **素材生成**：`dreamina text2image` 角色立绘 + 场景图
6. **视频生成**：AI 生成13维度提示词 → 门禁2扫描 → `dreamina multimodal2video` 提交（生成无口型视频）
7. **口型同步**：可灵/即梦对口型工具，将 TTS 音频贴到核心对白时间块
8. **字幕+拼接**：剪映加字幕气泡 → 叠加配音 → 合成最终合集

### 项目文件结构

```
项目根目录/
├── series_bible.md           ← 系列圣经（世界观·角色·章节结构）
├── book_rules.md             ← 项目规则（题材声明·外形锚点·台词规则）
├── current_focus.md          ← 当前创作焦点
├── project.json              ← 素材映射
├── timeline_truth.json       ← 时间线状态
├── asset_truth.json          ← 素材生成状态
├── gate_results/             ← 门禁结果
├── 素材/角色/ · 场景/        ← 角色立绘·场景图
├── 提示词/剧本/ · 视频/      ← 剧本·台词表·视频提示词
├── 输出/                     ← 视频成片
├── 音频/                     ← TTS配音音频 + 配音标注表
├── 帧图像/                   ← 首尾帧截图
└── 任务记录/                 ← submit_id 记录
```
